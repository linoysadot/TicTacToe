# TicTacToe
הפרויקט עוסק בפיתוח משחק "איקס-עיגול"(Tic-Tac-Toe)  בסביבה ממוחשבת, המאפשר לשחקן אחד או שניים לשחק בצורה אינטראקטיבית ונוחה. מטרת המערכת היא להדגים יישום של עקרונות תכנות מונחה עצמים באמצעות משחק מוכר, תוך יצירת חוויית משתמש ברורה ומהנה. המערכת פותרת את הצורך במשחק דיגיטלי נגיש, הכולל ניהול תקין של מהלך המשחק, בדיקת ניצחון, ושמירה על חוקי המשחק.
